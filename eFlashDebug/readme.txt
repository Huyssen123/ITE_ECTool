
It's for IT8987 eFlash Debug

ToolName     : eFlashDebug.exe
ToolVersion  : V08
Function     : This tool is for dump IT8987 eFlash Debug log
               and analysis binary log

Usage    : Please use administrator privileges to run

          1. Dump eFlash log
             eFlashDebug.exe /D
 
          2. Analysis binary log
             eFlashDebug.exe /A  abc.bin
             If there is no configuration file, The default configuration file will be created.
             You can add message code for configuration file
