This is an LED test method

1. Run LED_Test_M1.exe
2. Blink or force On LED one by one
   if LED blink you must press 'S'
   if LED force on you must press 'L'
3. Don't delete HZK16, LEDTest.cfg file
4. You can modify LEDTest.cfg to select support LED
5. You must modify cfg file name to LEDTest.cfg
